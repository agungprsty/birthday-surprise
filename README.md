# Digital Birthday Surprise

Sebuah kado digital interaktif berbasis web yang dirancang khusus untuk memberikan pengalaman kejutan ulang tahun yang berkesan. Proyek ini dibangun menggunakan **HTML5**, **Tailwind CSS**, dan **GSAP** untuk animasi yang halus.

## Fitur Utama
- **Interaktif:** Mulai dari input nama, amplop surat yang bisa dibuka, hingga kado yang melayang.
- **Auto-Personalization:** Mendukung URL Parameters untuk mengisi nama target secara otomatis.
- **Dynamic Title Case:** Mengonversi input nama menjadi format huruf kapital di setiap awal kata secara otomatis.
- **Voucher System:** Klik kado untuk mendapatkan hadiah "voucher" random yang bisa diklaim.
- **SEO Optimized:** Sudah dilengkapi Meta Tags untuk preview yang cantik saat dibagikan ke WhatsApp, Telegram, atau Instagram.

## Cara Penggunaan (URL Parameter)
Kamu bisa membagikan link dengan menambahkan parameter `?name=` di akhir URL agar nama istri langsung terisi otomatis.

**Contoh:**
`https://username.github.io/repository-name/?name=ahmad maliki`

Hasilnya di website akan otomatis menjadi: **Ahmad Maliki**

## Teknologi yang Digunakan
- [Tailwind CSS](https://tailwindcss.com/) - Styling & Layout.
- [GSAP (GreenSock)](https://greensock.com/gsap/) - Animasi transisi antar fase.
- [Canvas Confetti](https://www.npmjs.com/package/canvas-confetti) - Efek selebrasi perayaan.
- [Google Fonts](https://fonts.google.com/) - Menggunakan font *Dancing Script* & *Montserrat*.

## Struktur File
- `index.html`: Kode utama (Struktur, Style, & Logika).
- `thumbnail-preview.jpg`: Gambar preview untuk media sosial.
- `Jamrud - Selamat Ulang Tahun (Lirik).mp3`: Background music.
